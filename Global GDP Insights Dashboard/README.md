## Project Overview
This project merges and visualizes GDP (US$) and GDP growth (%) datasets to uncover economic patterns and insights at a global scale.  
It demonstrates skills in **data cleaning**, **data transformation**, and **Deployment - interactive visualization**.

### Key Features
- View **top countries by GDP** in any selected year  
- Explore **GDP growth trends** interactively  
- Filter by **year**, **country**  
- Built with **Dash**, fully deployed on **Render**  

---

## Data Preparation
Data used in this dashboard were processed using Python.  
Steps included:
1. Cleaning the World Bank GDP and GDP Growth datasets  
2. Merging both datasets on country code (`iso3`) and year  
3. Filtering to include only valid countries using the **pycountry** library  
4. Exporting a final tidy dataset (`Merged_tidy.csv`) for visualization
